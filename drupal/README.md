# Drupal Driver for NuoDB #

This interface is written in PHP for Drupal 8.x dev.

## PREREQUISITES ##

PHP 5.3.10
NuoDB PHP PDO Driver
NuoDB "drupal" schema

## BUILDING ##

NONE

## INSTALL ##

Recursively copy the nuodb directory into <path-to-drupal>/core/lib/Drupal/Core/Database/Driver.
